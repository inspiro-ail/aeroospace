Программный комплекс «Планетарный Классификатор» — это интеллектуальная система, предназначенная для анализа и автоматизированной классификации экзопланет с помощью искусственного интеллекта. Система оценивает физические и химические характеристики небесных тел, такие как масса, радиус, температура и состав атмосферы, чтобы определить их потенциальную полезность для человечества (колонизация, добыча ресурсов или наука).  
Ключевые особенности проекта:
• ИИ-анализ: использование локальной языковой модели llama3.2 через движок Ollama для формирования развернутых обоснований и оценки рисков.  
• Полная автономность: комплекс работает локально и без доступа к интернету, что гарантирует приватность данных.  
• Технологический стек: серверная часть на Python (Flask) и современный веб-интерфейс с интерактивной визуализацией результатов.  
• Гибкость данных: поддержка импорта и экспорта информации в форматах JSON и CSV.  
Система ориентирована на научно-исследовательские организации, образовательные учреждения и энтузиастов космоса.
